# Anchor Ports

Network port guide built with Astro and Tailwind CSS v4.

## Structure

- `astro.config.mjs`: GitHub Pages base path for `https://torifo.github.io/anchor-ports/`.
- `src/styles/global.css`: Tailwind v4 CSS-first theme, cyber chart grid, neon effects, strict borders, and reserved-area hatching.
- `src/data/ports.ts`: Main port ledger with sanctuary and collision-cost copy.
- `src/pages/index.astro`: First-screen usable port ledger.

## Commands

```bash
npm install
npm run dev
npm run build
```

GitHub Pages deploys from `.github/workflows/deploy.yml`.
